# Metal Forming Die Design – Cold Extrusion

## Overview
This project focuses on the design and analysis of a cold extrusion die system used to manufacture a collar bolt.

The work includes forming force calculations, die stress considerations, material selection, and CAD design of the tooling assembly.

## Objectives
- Design a cold extrusion process for a collar bolt
- Calculate forming forces and equivalent strain
- Design a reinforced die system with shrink ring
- Produce CAD models and technical drawings

## Engineering Approach
The process was analyzed using metal forming theory.

Key calculations include:
- equivalent strain
- mean flow stress
- extrusion force
- die stress and reinforcement using shrink fit

A full die set assembly was designed using CAD tools.
